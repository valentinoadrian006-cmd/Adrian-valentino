Tentang Saya

Saya membuat project ini dengan tema National Geographic karena saya tertarik dengan alam, satwa liar, dan eksplorasi. Website ini dibuat sebagai Final Project Pemrograman Web 1 menggunakan HTML, JavaScript dan Tailwind CSS.

Testimonial / Reflection

Selama mengikuti mata kuliah Pemrograman Web 1, saya belajar banyak hal baru terutama tentang bagaimana membangun website yang rapi, responsif, dan mengikuti standar modern. Saya juga belajar menggunakan Tailwind CSS yang membuat proses styling jauh lebih cepat dan efisien. Tantangan terbesar saya adalah memahami konsep responsif, tetapi setelah latihan dan mencoba banyak contoh, saya mulai terbiasa. Secara keseluruhan, kelas ini membantu saya memahami dasar pengembangan web dan membuat saya lebih percaya diri dalam membuat website sendiri.
